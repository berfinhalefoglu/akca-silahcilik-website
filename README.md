# Corporate Website and Admin Panel  
Developed for Akça Silahcılık

A web-based corporate website and management panel developed to present company information and manage basic content through an admin interface.

## Features
- Corporate website pages (Home, About Us, Services, Categories, FAQ)
- User membership and login page
- Admin panel for content management
- Backend server using Node.js
- Static frontend with HTML/CSS/JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js

## How to Run
1. Install Node.js
2. Open terminal in project folder
3. Install dependencies:
npm install

4. Start server:
node server.js

5. Open browser and go to:
http://localhost:3000
